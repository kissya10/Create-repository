# Orion Telegram Bot

Простой каркас проекта для игрового бота Orion на Python с aiogram 3 и SQLAlchemy.

## Установить зависимости

```bash
python -m pip install -r requirements.txt
```

## Настроить окружение

Скопируй `.env.example` в `.env` и заполни токен бота и другие переменные.

```bash
cp .env.example .env
```

## Запустить бота

```bash
python main.py
```

Бот запустится вместе с веб-сервером на http://localhost:8000.

## Основные команды

- `🐾 Магазин питомцев`
- `🐾 Мой питомец`
- `бонус`
- `профиль`
- `топ`
- `брак @username` или ответом на сообщение
- `варн` / `разварн` / `бан` / `кик` / `мут`
- `погода <город>`
- `id`
- `time`
- `Орион отправить [текст]` (в личке)
- `Орион титул [текст]`
- `Орион, стоит ли...` — космическое предсказание
- Нажми кнопку `📔 Обучение` в главном меню
- `босс` — рейд с фазами
- `угадай` — угадай слово
- `казино` — игра на монеты
- `🌐 Наш сайт` — ссылка на админку

## Веб-админка

Доступна на http://localhost:8080/admin. Только для SUPERADMIN_ID.

## Автоматические события

- Ежедневный босс в 12:00.
- Еженедельный топ в воскресенье 18:00.

## Структура проекта

- `database/` — модели и сессия SQLAlchemy
- `handlers/` — роутеры команд и логика бота
- `services/` — внешние сервисы (погода)
- `main.py` — запуск бота
